What is Reinforcement Learning?

Reinforcement Learning (RL) is a type of machine learning where an agent learns to make optimal decisions by interacting with its environment. Think of it like training a dog: the agent takes an action (like 'sit') and receives a **reward** (a treat) or a **penalty** (a "no"). Over millions of trials, it builds a 'policy'—a strategy to maximize its cumulative reward.

The 3D Web Environment

The "arena" in AIrena is a custom-built 3D world, streamed directly to your browser. It's powered by Three.js, the standard for 3D web graphics, which handles the rendering of the agent and the world. To make interactions realistic, the environment uses Ammo.js (a direct port of the Bullet physics engine) for collision detection and physics.

A New Challenge, Every Time

An agent that only masters one level isn't truly intelligent. To ensure the AI learns to *generalize*, AIrena's environment is procedurally generated for every training run using the "Rooms and Mazes" algorithm.

  • Placing random-sized rooms on the map.
  • Filling all remaining space with perfect mazes.
  • Finding all "connectors" (walls between two regions).
  • Opening connectors to merge all regions into one.
  • Removing dead-ends to create a more open map.

The Brain and The Body

AIrena runs as two connected parts. The "Brain" is a powerful Python agent using the Stable Baselines3 library to run algorithms like PPO. The "Body" is the 3D web visualizer. A WebSocket connects them, allowing the Python agent to send its actions (like 'move forward') and receive its 'senses' (like wall distance) to learn from.
